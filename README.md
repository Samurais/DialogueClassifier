# DialogueClassifier
通过分析客户和客服对话，对客户的问题进行一些分类。

## 实验方法
1. 基于SVM的分类方法
2. 基于Naive Bayes的分类方法
3. 基于LSTM循环神经网络的分类方法

## 数据集
约7200个对话文本，每个文本包含三级分类标签，其中一级分类有5类，二级分类有80类。

不同实验中，数据集的划分如下表所示：

|实验编号|训练集|验证集|测试集|
|:----:|:----:|:----:|:----:|
|1|80%|0|20%|
|2|80%|0|20%|
|3|60%|20%|20%|
